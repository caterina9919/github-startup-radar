## 操作說明

### 方式一：手動 star + GitHub Lists（推薦給想分類整理的情境）
1. 點開上表中的 GitHub 連結 → 點右上角「⭐ Star」
2. 點 Star 按鈕旁的下拉選單可將其加入自訂 List（例如建立「創業候選-Top10」「WantGoo參考」「Neuron Notes參考」三個 List）

### 方式二：GitHub CLI 批次加星（推薦給想一次處理 40 個的情境）
見同目錄 `star_all.sh`。需先安裝並登入 [GitHub CLI](https://cli.github.com/)（`gh auth login`），再執行：

```bash
bash star_all.sh
```

### 方式三：建立自己的「精選清單」倉庫
若想把本清單發布成公開的 awesome-list 型倉庫，可直接將本檔案內容作為新倉庫的 README.md。
